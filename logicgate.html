Circuit Code
Step 1: write code. Step 2: Check Errors. Step 3: Apply Code → UI.

Simple workflow

  1. Type or paste logic code on the left.
  2. Click Check Errors. If auto-fix is possible, the app will ask before changing your code.
  3. Click Apply Code → UI to create the visual circuit.
  4. Click UI → Code any time to turn your canvas back into portable code.

Most-used syntax

circuit "Half Adder"
input A = 0
input B = 1
gate SUM_LOGIC = XOR(A, B)
gate CARRY_LOGIC = AND(A, B)
output SUM = SUM_LOGIC
output CARRY = CARRY_LOGIC

Tips

Use short names like A, B, X1, SUM. Comments can start with # or //. Use place X1 240 80 or rotate X1 90 only when you need manual layout.

Help & Reference

      
    


    

      
      
      
      
    


    


      

        

          

            
🖱

            
Place Gates

            
Drag from the left panel onto the canvas, or press a quick key: A=AND, O=OR, N=NOT, X=XOR, I=INPUT, U=OUTPUT.

          

          

            
🔌

            
Draw Wires

            
Click an output pin (right side of a gate) and drag to an input pin (left side). Drag from an input pin to re-route an existing wire.

          

          

            
🔘

            
Toggle Inputs

            
Double-click an INPUT node to flip it 0↔1. Or select it and press Space. Use Tab / Shift+Tab to cycle through all inputs.

          

          

            
🔍

            
Navigate

            
Scroll to zoom. Space + drag or middle-mouse drag to pan. Press F to fit everything into view.

          

          

            
◻

            
Select & Move

            
Click a gate to select it. Drag on empty space to draw a selection box. Shift-click to add/remove from selection. Then drag to move.

          

          

            
🐛

            
Debug Panel

            
Press ` (backtick) to open the live signal monitor. All input/output/gate values update in real time. Click an input row to toggle it.

1️⃣

            
Create a Block

            
Place INPUTs, logic gates, and OUTPUTs on the canvas. Select them all (drag-select or Ctrl+A). Click Block in the toolbar or press Ctrl+B. Give it a name and color.

          

          

            
2️⃣

            
Use a Block

            
Saved blocks appear in the right panel. Drag them onto the canvas just like regular gates. The block's inputs/outputs become pins on the custom chip.

          

          

            
✏️

            
Edit a Block

            
Click Edit next to any block, or right-click a custom block gate and choose Edit Block. The editor enters "Sub-Circuit Mode" with full tools to tweak it.

          

          

            
📤

            
Export / Import

            
Click Export All to download your entire block library. You can also import multiple blocks at once by selecting several `.json` files via the Import... button.
